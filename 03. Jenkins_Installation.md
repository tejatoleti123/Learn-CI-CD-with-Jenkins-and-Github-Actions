# Jenkins Installation – Linux and Windows

This document provides **detailed Jenkins installation steps** for both Linux and Windows systems.

Always refer to the official Jenkins documentation for the latest updates:
https://www.jenkins.io/doc/

---

## Jenkins Prerequisites

- Java (JDK 11 or 17 recommended)
- Internet access
- Admin/root privileges

---

# Jenkins Installation on Linux (Ubuntu)

## Step 1: Update system packages

    sudo apt update

---

## Step 2: Install Java

    sudo apt install openjdk-17-jdk -y

Verify Java installation:

    java -version

---

## Step 3: Add Jenkins repository key

    curl -fsSL https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo tee \
    /usr/share/keyrings/jenkins-keyring.asc > /dev/null

---

## Step 4: Add Jenkins repository

    echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
    https://pkg.jenkins.io/debian-stable binary/ | sudo tee \
    /etc/apt/sources.list.d/jenkins.list > /dev/null

---

## Step 5: Install Jenkins

    sudo apt update
    sudo apt install jenkins -y

---

## Step 6: Start and enable Jenkins

    sudo systemctl start jenkins
    sudo systemctl enable jenkins

Check status:

    sudo systemctl status jenkins

---

## Step 7: Access Jenkins UI

Open browser:

    http://localhost:8080

Get initial admin password:

    sudo cat /var/lib/jenkins/secrets/initialAdminPassword

---

# Jenkins Installation on Windows

## Step 1: Install Java

- Download JDK from:
  https://adoptium.net/
- Install JDK 11 or 17
- Set JAVA_HOME environment variable

Verify:

    java -version

---

## Step 2: Download Jenkins

- Go to:
  https://www.jenkins.io/download/
- Download Windows installer (.msi)

---

## Step 3: Install Jenkins

- Run the installer
- Choose default settings
- Jenkins will be installed as a Windows service

---

## Step 4: Access Jenkins UI

Open browser:

    http://localhost:8080

Retrieve initial admin password from:

    C:\Program Files\Jenkins\secrets\initialAdminPassword

---

## Initial Jenkins Setup (Both OS)

- Enter admin password
- Install suggested plugins
- Create admin user
- Complete setup

---

## Important Notes

- Do not expose Jenkins publicly without security
- Use proper authentication
- Regularly update Jenkins and plugins
- Always refer to official documentation for changes
