# CI/CD with Jenkins and GitHub Actions – Hands-on Guide

This repository is designed for **Beginner to Intermediate Cloud & DevOps professionals** who want to learn **CI/CD concepts**, **Jenkins**, and **GitHub Actions** with **practical hands-on examples**.

## Who is this for?
- Beginners starting with DevOps
- Cloud Engineers (AWS / Azure / GCP)
- System Administrators moving into DevOps
- Anyone confused between Jenkins and GitHub Actions

## What you will learn
- CI/CD fundamentals
- Jenkins from scratch (installation → pipelines → agents)
- GitHub Actions from scratch (workflows → runners → secrets)
- Jenkins vs GitHub Actions comparison
- Real-world CI/CD projects
- Common interview questions
