This repository is created for Beginner to Intermediate Cloud / DevOps professionals who want to understand CI/CD concepts and learn Jenkins and GitHub Actions through clear explanations and hands-on practice.

The focus of this repository is to explain:
- Why CI/CD exists
- How CI/CD works in real projects
- How Jenkins and GitHub Actions are used in practice

This is not a shortcut-based repository. It is designed to build strong fundamentals.

---

## Why this repository?

Many beginners face these challenges:
- CI/CD concepts feel abstract
- Jenkins feels complex to start with
- GitHub Actions feels easy but unclear internally
- Confusion about when to use Jenkins vs GitHub Actions

This repository addresses those issues by:
- Starting from basics
- Explaining every concept in simple language
- Providing step-by-step hands-on guidance
- Following a structured learning flow

---

## Who is this repository for?

- Beginners starting with DevOps
- Cloud Engineers (AWS / Azure / GCP)
- System Administrators moving into DevOps
- Developers who want to understand CI/CD pipelines
- Anyone confused between Jenkins and GitHub Actions

Basic Linux and Git knowledge is expected.

---

## What you will learn

- CI/CD fundamentals (from scratch)
- Why CI/CD is important
- Jenkins architecture
- Installing Jenkins on AWS
- Jenkins Freestyle jobs
- Jenkins Pipelines
- How Jenkins fits into real-world DevOps workflows
